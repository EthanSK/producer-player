## Description

Producer Player helps music producers manage fast-moving mix and master exports in one desktop workflow.

What you can do:

• Track song versions automatically (v1, v2, v3) as exports accumulate
• Keep album song order locked and persistent across rescans
• Compare your mix against reference tracks with fast A/B switching
• Monitor loudness and peaks with integrated LUFS / true-peak analysis
• Preview platform normalization behavior (Spotify, Apple Music, YouTube, TIDAL)
• Add per-song checklist notes and ratings while listening
• Export latest ordered versions for handoff or release prep

Producer Player is designed for practical mastering and release preparation: fewer folder mistakes, faster comparisons, and cleaner final decisions.
