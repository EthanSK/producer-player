Producer Player

Hear every export version without folder chaos.

A desktop app for music producers who keep bouncing new passes and need one place to manage version history, audition references, and keep album order stable while making finishing decisions.

Why producers use it

Built around finishing workflows, not generic media-library features.

Group versions under one song

Keep v1, v2, v3 and beyond together so your folder doesn't turn into guesswork.

Mastering + reference checks in one place

Review LUFS/peaks, load a reference file, and quick-switch Mix vs Reference without leaving the app.

Keep order and export clean handoff folders

Lock your album sequence, then export latest versions in numbered order or move playlist order via JSON.
